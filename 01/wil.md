뷰포트 - 화면에 보이는 영역을 제어하는 기술, 미디어 쿼리로 화면 크기 감지해야 할 때 필요

플렉서블 박스 - 가변적인 박스 생성하는 기술 + 웹사이트의 구조 설계를 위한 기술

퀴즈
1. 반응형 웹 : PC, TV, 내비게이션, 스마트 기기 등 기기마다 또는 환경마다 최적화된 웹사이트를 제공하는 것

2. 호스팅 서버 : 인터넷 속 저장 공간

3. FTP 프로그램 : 호스팅 서버나 웹 서버에 파일을 자유자재로 옮기기 위한 프로그램 (ex : 파일질라)

4. 가변 그리드 : 웹사이트 제작 시 가변적으로 늘어나거나 줄어드는 구조 제작하는 기술, 가로 크기를 %로

5. 미디어 쿼리 : 컴퓨터에게 미디어의 종류 혹은 사이즈를 묻고 감지하여 웹사이트 변경

6. 뷰포트 : 화면의 보이는 영역을 제어하는 기술, 미디어 쿼리로 수없이 많은 화면의 크기를 감지해야 할 때 사용

7. 플렉서블 박스 : 가변적인 박스를 만드는 기술, 웹사이트의 구조 설계를 위한 기술
